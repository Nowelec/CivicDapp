INITIALISATION

Préparation du vote

1. Télécharger et déplacer le fichier vote-setup.json dans le répertoire Panel.

2. Ouvrir Panel/votes.html pour afficher les choix (QR code + Résultat).

Télécharger le fichier votants.json pour sauvegarder la liste des votants.

Comment créer une élection avec Civic ?

1. Donner un nom à l'élection et à l'organisateur

2. Remplir les choix et les votants

3. Cliquer sur le bouton « Générer ».

4. Attendre l'initialisation du vote.

5. Télécharger le fichier vote-setup.json sur votre ordinateur.
